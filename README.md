# Zomato Restaurant Analytics 🍽️

## Overview
Analyzed 148+ Zomato restaurants with 39K+ customer votes.
Interactive Power BI dashboard tracking ratings, costs, restaurant types & booking patterns.

## Dataset
- Restaurants: 148
- Customer Votes: 39K+
- Categories: Dining, Buffet, Cafes, Other
- Restaurant Types: Budget, Mid-Range, Premium
- Metrics: Ratings, Costs, Bookings, Reviews

## Tools Used
- Python - Data cleaning & EDA
- SQL - Restaurant rankings, aggregations
- Excel - Pivot tables
- Power BI - Interactive dashboard

## Power BI Dashboard
📊 KPI Cards:
- Total Restaurants: 148
- Total Votes: 39K
- Avg Rating: 3.63/5
- Avg Cost: ₹418.24

📈 Visualizations:
1. Top Rated Restaurants (Bar chart)
2. Restaurant Type Distribution (Pie)
3. Average Cost by Type (Bar)
4. Booking Patterns (Donut)
5. Online vs Offline (Slicer)

🎚️ Interactive Slicers:
- Country
- Restaurant Type
- Online/Offline
- Booked/Un-booked

## Key Insights
📌 Dining = 51.96% of votes (most popular)
📌 Mid-Range = 43.24% market share (sweet spot)
📌 Buffet = ₹871 avg cost (premium pricing)
📌 Rating avg = 3.63/5 (competitive market)
📌 Online ordering emerging trend
